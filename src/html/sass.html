Utvärdering

Starten

Det var relativt lätt att komma igång med SASS, då föreläsningarna var mycket givande med bra exempel. Jag började dock med att sätta upp alla partials enligt 7-1 metoden, men skalade ner allt eftersom och behöll endast de partials som jag tyckte passade för detta moment. Hade jag gjort om skulle jag börjat enkelt (såsom Malin visade) och sedan skala upp istället.

Fördelar:

  • Man kan skapa variabler som innehåller färgscheman, storlekar, font osv. Detta är kraftfullt då ändring på endast ett ställe slår igenom på all kod.
  • Användning av partials gör strukturen mer organiserad och ger en bättre överblick.
  • Mixins kan användas för att återanvända kod istället för att skriva samma kod många ggr.

Nackdelar:

  • För att kunna använda SASS krävs det en kompilator som gör om till vanlig CSS.
  • Möjligen kan koden bli lite komplicerad om man är glad i t.ex nesting osv.
