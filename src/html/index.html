Uppgiftsbeskrivning

Uppgiften är att skapa en demowebbplats som innehåller HTML, CSS och JavaScript. Sidan skall innehålla en huvudmeny med stöd för ett godtyckligt antal menyalternativ och vara responsiv.

Webbplatsen skall bestå av minst tre undersidor:

  • Startsida med info om uppgiften
  • Undersida för bilder med minst fem stycken fotografier
  • En undersida för publicering där du beskriver din skapade arbetsprocess.

Filerna skall vara väl strukturerade, med underkataloger för CSS, JavaScript och bilder.
